# ZPDS — 具身智能数据标准化管线

## 项目简介

ZPDS 是具身智能数据的标准化处理管线，目标是统一多个采集源（墨现、遁甲、简智新创、A2D 真机、EPIC-KITCHENS-100）的数据格式，经过质量检查、场景分割、标注后，产出可直接用于训练的标准 Prepared Segment。

## 分支说明

| 分支 | 内容 | 说明 |
|------|------|------|
| `main` | 项目骨架 | 完整的 `zpds/` 包结构、配置、测试框架 |
| `llccxx` | 功能探索 | 墨现数据清洗原型，独立验证后再合入 main |

> 两个分支基于不同提交分叉，文件互不重叠，互不干扰。

## 模块说明

### clean.py — 主程序入口

编排 10 步清洗管线，调用各模块完成检查并汇总结果。

```
python clean.py
```

**检查流程：**

| 步骤 | 检查项 | 负责模块 |
|------|--------|----------|
| 1 | 读取 meta.json | `reader.py` |
| 2 | 统计 index.jsonl 帧数 | `reader.py` |
| 3 | 帧数一致性比较 | `timestamp_checker.py` |
| 4 | 时间戳间隔检测（>40ms 异常） | `timestamp_checker.py` |
| 5 | 视频基本信息（帧数/FPS/分辨率） | `video_checker.py` |
| 6 | 坏帧检测（解码失败） | `video_checker.py` |
| 7 | 黑屏检测（灰度均值 < 5） | `video_checker.py` |
| 8 | 读取 IMU 数据 | `reader.py` |
| 9 | IMU 中断检测（>60ms 异常） | `imu_checker.py` |
| 10 | 生成清洗报告 | `report.py` |

### reader.py — 数据读取

| 函数 | 说明 |
|------|------|
| `read_meta(path)` | 读取 meta.json，提取 device/fps/分辨率/帧数/IMU 采样率 |
| `read_index(path)` | 读取 index.jsonl，提取 type=frame 行的时间戳列表 |
| `read_imu(path)` | 读取 imu_000000.csv，返回 DataFrame |

### timestamp_checker.py — 时间戳检查

| 函数 | 说明 |
|------|------|
| `check_frame_count(index_count, meta_count)` | 比较 index 与 meta 帧数是否一致 |
| `check_timestamp_gaps(timestamps)` | 检测时间戳间隔是否超过阈值（默认 40ms） |

### video_checker.py — 视频检查

| 函数 | 说明 |
|------|------|
| `check_video_info(path)` | 获取帧数/FPS/分辨率 |
| `check_bad_frames(path)` | 统计解码失败的坏帧数 |
| `check_black_frames(path)` | 检测灰度均值 < 5 的黑屏帧 |

### imu_checker.py — IMU 检查

| 函数 | 说明 |
|------|------|
| `check_imu_gaps(imu_df)` | 去重后检测 IMU 时间戳间隔超过 0.06s 的中断 |

### report.py — 报告生成

| 函数 | 说明 |
|------|------|
| `generate_report(results, path)` | 汇总所有检查结果，写入 clean_report.txt，返回 PASS/FAIL |

## 数据路径

当前数据集位于 `E:/datasets/egos/墨现/`，包含以下文件：

| 文件 | 说明 |
|------|------|
| `meta.json` | 会话元数据：设备/内参/IMU 偏置/schema 版本 |
| `index.jsonl` | 权威时间线：seq ↔ timestamp_ns 对应关系 |
| `color.mp4` | RGB 预览视频（1280×720@30fps，983 帧） |
| `depth.mp4` | 深度预览视频 |
| `color_000000.mkv` | RGB 原始容器（MJPG 编码） |
| `depth_000000.mkv` | 深度原始容器（Y16 编码） |
| `imu/imu_000000.csv` | IMU 时序（50Hz，ax/ay/az/gx/gy/gz） |
| `log/` | 录制日志（app.log 等） |

## main 分支 — 项目骨架（参考）

```
ZPDS/

│   ├── segmentation/           # Prepared Segment 生成
│   ├── prepared/               # 写入/修订/校验
│   ├── annotation/             # CEU 标注层

│   ├── release/                # Release 管理
│   └── utils/                  # 工具函数
├── configs/                    # 版本化配置
├── tests/                      # 测试
├── scripts/                    # 运行脚本
└── notebooks/                  # 调研 notebook
```
