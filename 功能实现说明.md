## 四、统一异常模型：QualityIssue

所有 5 个检测器的输出统一为此数据类：

```python
@dataclass
class QualityIssue:
    issue_type: str              # frame_count_mismatch | bad_frame | continuous_black_frames | timestamp_gap | imu_gap
    stream_id: str               # ego_rgb | ego_imu
    start_ns: int                # 异常起始 (设备时钟)
    end_ns: int                  # 异常结束 (设备时钟)
    severity: str                # warning | error | critical
    decision: str                # trim | split | keep_with_flag
    details: dict                # 检测器附加信息
```

### 5 种 issue_type 速查

| issue_type | 检测器 | 典型 decision |
|-----------|--------|--------------|
| `frame_count_mismatch` | `frame_count.py` | `keep_with_flag`（差异≤5 → warning，>5 → error） |
| `bad_frame` | `bad_frame.py` | `keep_with_flag`（标记，不切分） |
| `continuous_black_frames` | `black_frame.py` | 首尾 → `trim`，中间长 → `split` |
| `timestamp_gap` | `timestamp_gap.py` | ≥0.5s → `split`，<0.5s → `keep_with_flag` |
| `imu_gap` | `imu_gap.py` | ≥1.0s → `split`，<1.0s → `keep_with_flag` |

## 五、候选 Segment 生成算法

```
输入: issues (list[QualityIssue]), session_start_ns, session_end_ns

1. 分离 issue:
   - decision = "trim" 且靠近起始 → head_trims
   - decision = "trim" 且靠近末尾 → tail_trims
   - decision = "split" → split_boundaries

2. 计算有效范围:
   valid_start = max(session_start, 最晚 head_trim.end)
   valid_end   = min(session_end,   最早 tail_trim.start)

3. 切分:
   按 split_boundaries 中的缺口切割
   [valid_start, gap1.start], [gap1.end, gap2.start], ...

4. 过滤:
   - 丢弃 duration < min_duration_s (默认 1s)
   - 标记 duration > max_duration_s (默认 120s)
```

## 六、Prepared Segment 产出结构

```
prepared_segments/seg_000001/
├── segment.json                    # 核心控制文件
│   ├── segment_id, source_session
│   ├── source_assets (4个源文件 SHA-256)
│   ├── timeline (相对时钟: 0 → duration_ns)
│   ├── source_span (设备时钟范围)
│   ├── streams[ego_rgb + ego_imu]
│   ├── calibration_uri
│   └── quality (status + issues，从 candidates 传入)
├── data/
│   ├── ego_rgb.mp4                 # H.264 CFR 30fps, 1280×720
│   └── imu.parquet                 # 相对时间 + 标准列名
├── maps/
│   └── rgb_sample_map.parquet      # 输出帧↔源帧 最近邻映射
├── calibration/
│   └── calibration.json            # 针孔模型内参 + 外参
└── reports/
    └── validation.json             # 6项自动校验
```
